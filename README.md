# CarHub Dealer Inventory Manager

## Description

CarHub is a React and Supabase application that allows customers to browse vehicle inventory and submit inquiries while allowing authenticated dealers to manage vehicle listings.

## Features

- Vehicle inventory browsing
- Vehicle details page
- Customer inquiry submission
- Dealer authentication
- Protected admin dashboard
- Add vehicle
- Update vehicle status
- Delete vehicle
- SCSS styling structure

## Technologies

- React
- React Router
- Supabase
- SCSS
- JavaScript

## Installation

npm install

npm run dev

## User Flow

Customer:
Browse Vehicles → View Details → Submit Inquiry

Dealer:
Login → Manage Inventory → View Inquiries
