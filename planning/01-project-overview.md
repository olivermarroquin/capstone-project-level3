# CarHub Dealer Inventory Manager

## Project Goal

Create a React and Supabase application that allows a car dealership to manage vehicle inventory while allowing customers to browse vehicles and submit inquiries.

## User Types

### Customers

- View available vehicles
- View sold vehicles
- View vehicle details
- Submit inquiries

### Dealers

- Sign in
- Add vehicle listings
- Update vehicle status
- Delete vehicle listings
- View customer inquiries

## Main Technologies

- React
- React Router
- Supabase Database
- Supabase Authentication
- SCSS
- JavaScript

## Core Features

### Vehicle Inventory

Customers can browse all dealership inventory.

### Customer Inquiries

Customers can submit inquiries about a specific vehicle.

### Dealer Dashboard

Dealers can manage inventory and view inquiries.
